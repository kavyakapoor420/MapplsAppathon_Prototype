# Installation steps of our Project 

1. Clone the repository:
   ```bash
   git clone  https://github.com/kavyakapoor420/MapplsAppathon_Prototype.git Website
   ```
   

2. then  
    ```
     cd Website 
    ```
3. npm install
4. npm run dev
5. Open your browser and visit:   http://localhost:5173
